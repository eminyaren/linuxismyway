Bash script for ffmpeg

2020-11-25 — Written by Emin Yaren

Using Ffmpeg is very effective way to convert video files for Youtube and Davinci Resolve. If you want to convert more than one video files a Bash script is very useful.

Let’s create a converting script

I am not a programmer but some times I need to create some scripts. So the script which we will create can be different that we create.

First, we need to decide some variables:

  • Source files location: srcDir
  • Source files extensions: srcExt
  • converting type (ProRes or Youtube): type
  • Destination location: destDir
  • Destination extention: destExt
  • Options for ProRes Conversation: opts

Now create “convert” script (I am using vim, you can use any text editor):

> ~$ vim convert

Prompt user and read some variables:

#!/bin/Bash
  • We ask user for desired file type
echo “Please select convert type:”

echo “1 for Prores, 2 for Youtube”
  • Reading user input to type variable
read type
  • We ask user for source extensions
echo “Enter Source Extension”
  • Reading user input for srcExt variable
read srcExt
  • We ask user to location of files
echo “Enter location of files:”
  • Reading user input for srcDir variable
read srcDir

we have what we need, we can create real script now.

  • Enter to source directory
cd $srcDir
  • We use “IF condition” to understand what user choose. If user choose “ProRes” it will execute these commands
if [ $type -eq “1” ] ; then
  • Desired extension will be “mov”
destExt=“mov”
  • Create “mov” folder for converted files and create “destDir variable”
mkdir mov

destDir="./mov"
  • Now we need to know ProRes options so We ask user for it.
echo “Prores options (Proxy=0, lt=1, standart=2, hq=3, Pr4444=4, Pr4444xq=5)”

read opts
  • We will create a for loop to convert every file in the directory
 for filename in “$srcDir”/*."$srcExt"; do
  • These two lines remove source file extension
 basePath=${filename%.}
 baseName=${basePath##/}
  • Next step is Ffmpeg, this line will run for every file in the directory
ffmpeg -i “$filename” -c:v prores_ks -profile:v $opts -c:a pcm_s16le “$destDir”/"$baseName"."$destExt"
  • Then we will close “for loop” and “if condition”.
done fi
  • Now We will create a new condition and loop for Youtube option. We can use same code with some difference.
if [ $type -eq “2” ] ; then
destExt=“mp4”
destDir="./mp4"
mkdir mp4
for filename in “$fileDir”/."$srcExt"; do
basePath=${filename%./}
baseName=${basePath##/}
ffmpeg -i “$filename” -c:v libx264 -b:v 8192k -b:a 192k “$destDir”/"$baseName"."$destExt"
done fi

It is time to save our batch file. After that we need to give executable permissions to the file. In the command line run this command:

    ~$ chmod +x convert

We can copy the file in “/usr/local/bin” so we can run our script in every folder.

Happy Bash scripting!..
